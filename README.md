# RPG da Vida

Pacote pronto para desenvolvimento com Vite + React + Tailwind.

## Como usar

1. Instale dependências:

```
npm install
```

2. Rodar em modo dev:

```
npm run dev
```

3. Build para produção:

```
npm run build
```

Os sprites são placeholders SVG em `public/sprites`. Substitua por arte PNG para melhor visual.
